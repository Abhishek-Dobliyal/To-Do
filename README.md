## Installation and Dependencies

1. [Python](https://www.python.org)
2. [NodeJS](https://nodejs.org/en/)

## Frontend

### Project setup
```
npm install vue
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
